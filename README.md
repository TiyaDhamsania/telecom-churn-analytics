# 📊 Telecom Customer Churn Analytics Dashboard

![Dashboard Preview](page1_executive_overview.png)

## 🎯 Project Overview
End-to-end churn analytics solution analyzing **7,032 telecom 
customers** to identify $1.67M revenue at risk and recommend 
data-driven retention strategies.

## 🛠️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| Python | EDA, Data Cleaning |
| Pandas, Seaborn, Matplotlib | Analysis & Visualization |
| Scikit-learn | Logistic Regression, Random Forest |
| SQL | Data Querying & Analysis |
| Power BI | Interactive Dashboard |

## 📈 Key Business Findings
- Overall churn rate: **26.58%** — 6.5% above industry benchmark
- Annual revenue at risk: **$1.67M**
- Month-to-month customers churn at **42.7%**
- Electronic check users churn at **45.3%**
- Fiber optic users churn at **41.9%**
- High risk segment: **976 customers** at **67.7% churn rate**

## 🤖 ML Model Performance
| Model | Accuracy | Precision | Recall | F1 Score |
|-------|----------|-----------|--------|----------|
| Logistic Regression | 77.9% | 70% | 30% | 42% |
| Random Forest | 78.5% | 65% | 43% | 51% |

**Random Forest selected** — higher recall means better 
at identifying actual churners before they leave.
